Assignment 4: note

  • Title of the PDF is wrong (should be "Assignment 4")
  • Calculate the norm as well, like in Assignment 2 and 3
  • Use the same variants as for Assignment 2 and 3
  • Compile and run on cscluster with GSL CBLAS from /home/cs/bobr.
    To your source code add this:
    #include <gsl/gsl_cblas.h>
    Compile with:
    $ mpicc -I/home/cs/bobr/include -L/home/cs/bobr/lib -lgslcblas ...

MxM application:

  1. Allocate and initialize full matrices on root (0)
  2. Distribute matrices with collective communication operations
  3. Compute in parallel
  4. Gather result on root

Norm:

  1. Finding max at each process
  2. Reduce MAX to root

Column-based matrix partitioning only (2-b):

  • Tutorial on MPI derived data types
  • Example:

      MPI_Datatype col_type;
      MPI_Type_vector(myrows, b, pivcol_ld, MPI_DOUBLE, &col_type);
      MPI_Type_commit(&col_type);
      MPI_Isend(pivcol, 1, col_type, (col_rank + 1) % params->npcols, 0, comm_row, &reqs[count]);
      MPI_Type_free(&col_type);
